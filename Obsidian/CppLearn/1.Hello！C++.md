## 第一部分  "Hello World!"

在本 markdown 同目录下你会看到一个`HelloWorld.cpp`

```c++
#include <iostream>

int main(){
    std::cout << "Hello World!" << std::endl;
    return 0;
}
```

同目录下的`HelloWorld.exe`为本代码文件的g++编译结果

### 如何编译？

前提是你的 windows 中配置了 MinGW/g++ 的全局环境变量，否则是无法编译的。

```bat
g++ HelloWorld.cpp -o HelloWorld.exe
start HelloWorld.exe
```
