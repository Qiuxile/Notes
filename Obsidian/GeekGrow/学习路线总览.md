## 学习路线总览

| 阶段  | 核心内容                                          | 产出                       | 学习目标                                      | 相关项目                             | 未来适应方向与AI应用                           |
| :-- | :-------------------------------------------- | :----------------------- | :---------------------------------------- | :------------------------------- | :------------------------------------ |
| 0   | 极客工具链（[[Linux]], [[Git]], [[Vim]], [[Shell]]） | 熟练命令行，脚本自动化              | 掌握命令行操作、文件操作，了解 Git 工作流程，使用 Shell 完成自动化任务 | 编写文件管理脚本，Git 项目管理，自动化部署          | 探索 AI 辅助开发工具（如 GitHub Copilot），提升效率   |
| 1   | Python（第一语言）                                  | 自动化脚本、爬虫、简单 Web          | 熟悉 Python 语法，掌握数据结构与算法的基础，能够编写简单的 Web 应用  | 爬虫项目、自动化工具、Flask Web 应用          | 学习使用 AI 工具进行代码自动化，快速生成代码和解决方案         |
| 2   | C（内存与指针）                                      | 动态数组、哈希表、简易 Shell        | 深入理解内存管理，掌握指针，了解静态与动态内存分配的区别              | 实现链表、动态数组、哈希表，简易命令行工具            | 探索 AI 驱动的编程模型优化工具和代码性能分析工具            |
| 3   | C++（面向对象、泛型、RAII）                             | 手写 String、Vector、贪吃蛇、内存池 | 掌握 C++ 的面向对象特性，理解 RAII 原则，掌握 C++ 标准库      | 实现自定义容器（String、Vector 等），开发贪吃蛇游戏 | 掌握与 AI 相关的并发编程和大数据处理技巧                |
| 4   | JavaScript 核心速成（为 TS 打底）                      | 理解 this、原型、闭包、事件循环       | 熟悉 JavaScript 的基础知识，掌握异步编程和事件驱动模型         | 实现一个简单的 Todo List，手动实现事件循环       | 深入学习 JavaScript 在前端开发中的 AI 应用，如智能推荐算法 |
| 5   | TypeScript + 全栈（前端 + Node + 数据库）              | 完整博客系统（前后端），部署上线         | 掌握 TypeScript 的类型系统，深入了解前后端开发流程，熟悉数据库操作   | 全栈博客系统，用户认证、文件上传功能实现             | 学习使用 AI 和自动化工具提升全栈开发效率，利用云服务平台部署应用    |
| 6   | Go（并发与网络服务）                                   | 并发文件服务器、聊天室              | 理解 Go 的并发模型，掌握高效的网络编程，能够开发高性能的网络应用        | 构建聊天室，开发文件上传和下载服务                | 探索基于 Go 的高效并发和 AI 协同应用，如智能服务调度        |
| 7   | 持续进阶（算法、系统设计、C++ 深度项目）                        | 根据兴趣选择（见进阶清单）            | 根据个人兴趣深入系统设计、数据结构与算法或其他领域                 | 参与开源项目，设计分布式系统或性能优化项目            | 探索 AI 在系统设计、自动化部署与优化中的应用，学习量子编程等前沿技术  |
