## 常见问题

### Q: 舵机不转动怎么办？
1. 检查电源连接
2. 确认信号线连接正确
3. 检查控制信号是否正确
4. 测试舵机是否损坏

### Q: 如何选择合适的舵机？
1. 根据扭矩需求选择
2. 考虑转速要求
3. 注意工作电压
4. 查看尺寸和重量限制

### Q: 编程时需要注意什么？
1. 信号频率要匹配舵机要求
2. 角度范围不要超过极限
3. 添加适当的延迟
4. 错误处理机制
